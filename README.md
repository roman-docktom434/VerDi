СРЕДА ЗАПУСКА
------------
1) Web-сервер с поддержкой PHP интерпретации (Apache/ Nginx или встроенный PHP сервер);
2) Требуется установленная Java;
3) требуется установленный MySQL Server (версия 8+);
4) (Опционально) MySQL Workbench для управления БД.


УСТАНОВКА
------------
### Установка проекта VerDi

Выполните 
~~~

git clone https://github.com/roman-docktom434/VerDi.git

~~~
### БАЗА ДАННЫХ

Проект использует СУБД MySQL.

1. Запустите MySQL сервер

2. Создайте базу данных:

~~~
CREATE DATABASE diploma_system;
~~~

3. Выберите базу данных:

~~~
USE verdi;
~~~

4. Импортируйте структуру базы данных:

Если у вас есть файл `database.sql`, выполните:

~~~
mysql -u root -p verdi < Backend/database.sql
~~~

ИЛИ вставьте SQL-скрипт вручную через MySQL Workbench.

5. Настройте подключение к базе данных:

Откройте файл:

~~~
Backend/db.php
~~~

и укажите параметры подключения:

~~~
$host = "localhost";
$dbname = "verdi";
$user = "root";
$pass = "ваш_пароль";
~~~
### Выполнение миграций

В данном проекте миграции представлены в виде SQL-скрипта (`database.sql`).

Для инициализации базы данных:

1. Убедитесь, что база `verdi` создана
2. Выполните SQL-скрипт:

~~~
mysql -u root -p diploma_system < Backend/database.sql
~~~

После выполнения будут созданы таблицы:
- Student
- HR
- HighSchool
- Diplom
